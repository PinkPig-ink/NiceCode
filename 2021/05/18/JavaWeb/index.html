1、 基本概念

1.1 前言

web开发：

  • web、网页的意思

  • 静态web资源

    • 技术：html、css
    • 提供给所有人浏览的数据始终不会发生变化

  • 动态web资源

    • 指web页面中供人们浏览的数据是由程序产生的，不同时间点访问web页面看到的内容各不相同

    • 技术栈：JavaScript、JSP/Servlet、ASP、PHP等

Java Web

​ 在Java中，动态web资源开发技术统称为Java Web。
