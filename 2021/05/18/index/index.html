1.github创建一个项目

创建项目时勾选 使用md文件初始化，记得去选择一个好看的适合你的主题。

2.安装hexo博客搭建框架

安装的前提是你已经安装了

  • node.js
  • Git

具体的安装方法大家可以官方文档看。

在你的终端运行

npm i hexo-cli -g

等待安装完成。

hexo -v

查看是否安装成功。

这样就可以了。

然后就要初始化我们的网站，输入hexo init初始化文件夹，这个文件夹必须是空的。

接着输入npm install安装必备的组件。

这样本地的网站配置也弄好啦，输入hexo g生成静态网页，然后输入hexo s打开本地服务器，然后浏览器打开http://localhost:4000/

然后简单填写一下

输入ssh -T git@github.com，再回答yes，如果如下图所示，出现你的用户名，那就成功了。

打开博客根目录下的_config.yml文件，这是博客的配置文件，在这里你可以修改与博客相关的各种信息。

修改最后一行的配置：

repository修改为你自己的github项目地址。

4.写文章、发布文章

首先在博客根目录下右键打开git bash，安装一个扩展npm i hexo-deployer-git。

然后输入hexo new post "article title"，新建一篇文章。
